How it works: Pick the monthly stock-report .xlsx, set the date you want to export. The engine fills ST Data, Raw Data, and Summary Data sheets while preserving 100% of the workbook's original formatting and formulas. The source file is never modified — a new copy is saved next to it.
  • ST Data — daily summary row per tank/material (Cavern net, TK transfer in/out, GI Tanlorry) — built from WMS ST + TL Data.
  • Raw Data — appends every TL Data row for the selected GI date.
  • Summary Data — groups by customer / trade / type and fills into the template's pre-made slot rows.
  • Check Data runs the pre-export validation: WMS GI ↔ TL Data, GI by tank, SAP cross-check (5 days + export date).

📐 %wt C3 — standalone update

Updates %wt C3 only. Does not change the entered opening C3/C4 stock. Applies to every C3/C4 split calculation from this point on (logged to history).

📥 Opening Stock

This is the day's baseline. Current stock = Opening + cavern in + transfers − exported (TL). Overwrites any previous opening stock for this tank & date.

⇄ Cavern receive / return

Positive = received from cavern into the tank. Negative = returned to the cavern.

⇆ Inter-tank transfer

Deducted from the source tank, added to the target tank. Recorded as one matched pair (one history row per tank).
